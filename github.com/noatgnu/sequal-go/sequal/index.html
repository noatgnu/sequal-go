type AminoAcid

type AminoAcid struct {
	BaseBlock
	// contains filtered or unexported fields
}

AminoAcid represents an amino acid residue with its position, modifications, and mass properties. It embeds BaseBlock to inherit common functionality and maintains a list of modifications that can be applied to the residue.

func NewAminoAcid

func NewAminoAcid(value string, position *int, mass *float64) (*AminoAcid, error)

NewAminoAcid creates a new AminoAcid instance with the specified value, position, and optional mass. If the amino acid is not recognized and no mass is provided, it returns an error. The mass parameter overrides the default mass for known amino acids.

Example: 

// Create a new amino acid 'P' at position 0
pos := 0
aa, err := sequal.NewAminoAcid("P", &pos, nil)
if err != nil {
	fmt.Println("Error:", err)
}

fmt.Println(aa.GetValue()) // "P"
